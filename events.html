More jQuery Event Handling

  • 'blur' - this event is emitted when an element loses focus, which happens when the user has focus on an element and Tabs to another element or clicks elsewhere
  • 'change' - this event is emitted when the value of an input field is changed, such as a text area, text input field, radio button, etc.
  • 'click' - this event is emitted when an element is clicked on
  • 'dblclick' - this event is emitted when an element is double-clicked on
  • 'error' - this event is emitted when an element is not loaded correctly, such as elements
  • 'focus' - this event is emitted when an element gains focus, meaning that the user has reached the element via the Tab key or has clicked on it
  • 'hover' - this event is emitted when the mouse pointer enters or leaves the element
  • 'keydown' - this event is emitted when the user first presses any key down
  • 'keyup' - this event is emitted when the user releases a key
  • 'mousedown' - this event is emitted when the user presses a mouse button over the targeted element
  • 'mouseup' - this event is emitted when the user releases a mouse button over the target element
  • 'mouseenter' - this event is emitted when the user moves the mouse pointer into the targeted element
  • 'mouseleave' - this event is emitted when the user moves the mouse pointer out of the targeted element
  • 'mousemove' - this event is emitted when the user moves the mouse around within the targeted element
  • 'resize' - this event is emitted when the targeted element is resized
  • 'scroll' - this event is emitted when the target element is scrolled
  • 'select' - this event is emitted when the target element is selected, usually only used for various types of inputs
  • 'submit' - this event is emitted when the target element is submitted, and can only be attached to form inputs
